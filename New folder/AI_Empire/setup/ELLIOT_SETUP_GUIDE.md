## STEP 6 — Webhook Setup (Auto-Track Payments)

When running your server publicly, replace `YOUR_SERVER_URL` with your actual domain or ngrok URL:

**For testing locally:**
```bash
npx ngrok http 3000
```
This gives you a public URL like `https://abc123.ngrok.io` — use that for webhooks.

**PayPal webhook:**
1. developer.paypal.com → My Apps → Your App → Webhooks
2. Add URL: `https://YOUR_URL/webhooks/paypal`
3. Events: PAYMENT.CAPTURE.COMPLETED, INVOICING.INVOICE.PAID

**Wise webhook:**
1. wise.com → Settings → Developer tools → Webhooks
2. URL: `https://YOUR_URL/webhooks/wise`
3. Event: transfers#state-change

**Gumroad:**
1. app.gumroad.com → Settings → Advanced → Ping URL
2. URL: `https://YOUR_URL/webhooks/gumroad`

**LemonSqueezy:**
1. app.lemonsqueezy.com → Settings → Webhooks
2. URL: `https://YOUR_URL/webhooks/lemonsqueezy`
