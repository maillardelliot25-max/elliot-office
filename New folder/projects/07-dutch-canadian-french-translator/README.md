# Dutch → Canadian French Translator

Cultural localization with Montréal/hockey/poutine context.

## Features

- **Canadian Context** - Hockey, poutine, Montréal references
- **Natural Translation** - Conversational Canadian French
- **Single Direction** - Dutch input, French output focus
- **Smart Caching** - Repeated translation cache

## Tech

Node.js, Express, Claude API, NodeCache

## Usage

Run `INSTALL.bat` and open http://localhost:5300
