# German → Québécois French Translator

One-way translation with Québécois cultural variant support.

## Features

- **Québécois Awareness** - tu, c'est, y'a instead of vous
- **Cultural Localization** - Québécois-appropriate expressions
- **Read-only Output** - Focus on translation result
- **1-hour Cache** - Fast repeated translations

## Tech

Node.js, Express, Claude API, NodeCache

## Usage

Run `INSTALL.bat` and open http://localhost:5200
