# SaaS Metrics Visualizer

Real-time SaaS metrics dashboard with HTML5 Canvas data visualization.

## Resume Bullets

• **Built** SaaS metrics visualization dashboard with HTML5 Canvas; MRR, churn, CAC, and LTC tracking

• **Implemented** responsive multi-chart system (line and bar charts) with real-time metric updates

• **Designed** simulation engine with month-by-month growth projection and autoplay controls

• **Tech:** HTML5 Canvas API, CSS3, vanilla JavaScript (zero dependencies)

## Usage

Open `index.html` in any browser. Click "Simulate Month" to project growth, or "Autoplay" for continuous simulation.
