# Maillard AI - Autonomous Revenue System

Real-time autonomous agent dashboard with live revenue tracking and task execution.

## Features

- **6 Autonomous Agents** - LinkedIn, Social Media, Freelance, AI Dev, Email, Consulting
- **Real-time Updates** - Socket.IO for live revenue and task tracking
- **Agent Control** - Execute tasks on-demand with one-click activation
- **Revenue Dashboard** - Total daily revenue with per-agent metrics
- **Dark Theme** - GitHub-style interface with cyan accents

## Tech

Node.js, Express, Socket.IO, HTML5, CSS3, vanilla JavaScript

## Usage

Run `INSTALL.bat` and open http://localhost:5000
