# Spanish ↔ English Translator

Bidirectional translation with Claude API and NodeCache caching.

## Features

- **Split-pane UI** - Spanish on left, English on right
- **Bidirectional** - Translate either direction
- **Smart Caching** - 1-hour TTL cache for repeated translations
- **Language Detection** - Automatic source language identification
- **Swap Button** - Quick pane swap functionality

## Tech

Node.js, Express, Claude API, NodeCache, HTML5, CSS3

## Usage

Run `INSTALL.bat` and open http://localhost:5100
