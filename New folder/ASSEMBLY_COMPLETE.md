## Support & Troubleshooting

**Issue**: Agent not reporting revenue
- Check `.env` configuration
- Verify FINANCE_AGENT_URL is correct
- Review agent logs: `tail -f logs/aria.log`
- Test webhook manually: `curl -X POST http://localhost:5000/api/events/revenue`

**Issue**: Portfolio not loading
- Verify GitHub Pages enabled
- Check branch is `main` (not `master`)
- Clear browser cache
- Test locally: `open portfolio-website/index.html`

**Issue**: High CPU usage
- Check for infinite loops in agent code
- Monitor process: `top -p <PID>`
- Profile with Node.js profiler
- Scale to multiple instances

**Issue**: Data discrepancies
- Run revenue reconciliation: `npm run reconciliation`
- Compare reported vs actual
- Check Finance Agent logs
- Restore from backup if needed
