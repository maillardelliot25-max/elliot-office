## **Add Real Money**

### **Option 1: Log Income Manually**
1. Click 💳 **Income Log**
2. Enter amount (e.g., $100)
3. Choose source (Fiverr, Upwork, Wise, bank transfer, etc.)
4. Click ✅ Log Payment
5. Shows in dashboard instantly

### **Option 2: Auto-Track Webhooks**
1. Click ⚙️ **Settings**
2. Scroll to "Payment Accounts"
3. Enter your **Wise email** + **PayPal email**
4. When you get paid → dashboard updates automatically

### **Option 3: Add Your Card (Linx)**
1. Click ⚙️ **Settings**
2. Scroll to "Add Your Credit Card (via Linx)"
3. Enter card details
4. Click ✅ Add Card via Linx
5. Ready to receive payouts

## **Stop the Server**

**Option 1:** Close the black `RUN_FOREVER.bat` window

**Option 2:** Double-click `STOP.bat`

**Option 3:** Open Command Prompt, type:
```bash
taskkill /F /IM node.exe
```

## **Access on WiFi (Other Devices)**

On any device on your WiFi:
1. Open browser
2. Go to: `http://[YOUR_COMPUTER_IP]:3000`
   - Example: `http://192.168.1.100:3000`
3. Login with `admin@empire.ai` / `empire2024`

To find your IP:
- Windows: Open Command Prompt, type `ipconfig`, look for "IPv4 Address"

## **Troubleshooting**

| Issue | Fix |
|-------|-----|
| Port 3000 in use | Double-click `STOP.bat`, wait 5s, try again |
| Can't login | Check password is exactly `empire2024` |
| No agents running | Click ✈️ **Autopilot** → Check "Autopilot: ON" |
| Missing data | Restart `RUN_FOREVER.bat` |
| Slow dashboard | Refresh browser (Ctrl+R) |
